# Excel_Deemah_Store_Analysis
Deemah Store Annual Report 2026 - Interactive Dashboard
This repository contains an interactive sales analysis dashboard for Deemah Store, created using Microsoft Excel. The project provides a comprehensive overview of the store's performance throughout 2026, focusing on sales trends, customer demographics, and order fulfillment.

üìä Dashboard Overview
The dashboard is designed to help stakeholders visualize key business metrics and make data-driven decisions. It includes multiple visualizations that track everything from monthly revenue to regional performance.

Key Visualizations:
Order vs Sales (Monthly Trend): A combination chart showing the relationship between the total count of orders and the total sales amount from January to December.

Sales: Men vs Women: A pie chart illustrating the gender-wise distribution of sales revenue.

Order Status: A detailed breakdown of order fulfillment (Delivered, Refunded, Returned, Cancelled).

Sales by Top 5 States: A horizontal bar chart identifying the highest revenue-generating regions (e.g., Madhya Pradesh, Bihar, Maharashtra).

Order: Age vs Gender: A clustered column chart analyzing order distribution across different age groups (Adult, Senior, Teenager) segmented by gender.

Order by Channel: A visualization showing the contribution of different sales channels (Flipkart, Amazon, Myntra, etc.).
